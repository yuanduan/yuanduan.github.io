游戏开发记录-#0.1 重命名新建文件夹

发表于 2020-07-23 | 分类于 游戏开发

才过了一天，我就准备更新一下日志了，过于敬业。

这一期日志更新的方向是有关营地养成要素的。

模式切换类卡牌的获取

在上一篇中原暂定的打怪获取方式被否决了，不过这也是必然的。

以及，为了称呼的方便，将在之后的文档记录中称呼为武器卡。

为了增加长线养成的乐趣，讨论后决定将武器卡和直接增益类卡牌放入到[营地]这部分系统中去。

将卡牌的直接获取方式与营地的建设挂钩，提高游戏整体的可玩性与重复游戏性。

具体的获取方式则为引入一个货币的概念，使用这种货币进行开卡包，抽取复数张武器和直接增益卡和消耗品卡（暂定，或许会在卡池里加入其它道具）。有关这部分内容我会放到下一节去讲述。

营地建设

在地牢中的探索中，会获得复数被动增益卡，这些卡会在一次地牢探险后结算成一种货币。

货币直接关系到营地内的各种建筑的搭建和升级，以及玩家box的提升。

现预定的建筑有如下几种：

*铁匠铺：负责升级和修改武器卡的词缀、属性等要素。以及直接增益卡的合成；
*商店：出售一些简单的消耗品卡，等级提升后会偶尔出售些稀有道具；
*卖卡包的（名称未定）：出售卡包，提升等级会解锁更多的卡，以及更容易抽到稀有卡牌。设计中同一种武器卡有着不同的[切换动作]，[切换动作]的稀有度；
*塔罗屋：强化[塔罗牌]这一装备要素，根据建筑等级决定塔罗的升级上限。预定每种塔罗都会有不同程度的改变游玩风格的设定。预定把塔罗升级相关素材放到卡包里。；
